# 📚 ads-estudos

Repositório destinado ao armazenamento de exercícios, projetos e anotações realizados por meio de estudos à parte do curso. O foco deste repositório é a prática constante e a documentação da minha evolução técnica.

---

## 🛠️ Tecnologias e Ferramentas
* **PHP** (Back-end e lógica de servidor)
* **HTML5 & CSS3** (Estruturação e estilização)
* **Git & GitHub** (Versionamento e organização)

---

## 📂 Organização do Repositório

O conteúdo está dividido por tecnologias e níveis de complexidade:

### 🐘 PHP
Nesta seção, documento o aprendizado de lógica de programação, manipulação de dados e integração com o servidor.

* **`php/exercicio-post`**: Sistema para prática de envio de dados via formulários. Focado no uso da superglobal `$_POST`, sanitização de dados e exibição dinâmica de resultados.
* **`php/exercicio-get`**: (Exemplo) Prática de parâmetros via URL utilizando a superglobal `$_GET`.
* **`php/logica-algoritmos`**: (Exemplo) Exercícios focados em estruturas de repetição (for, while), condicionais e manipulação de arrays.
